Introducing RoboShellGuard: The Evolution of ShellGuard / The ShellGuard project has officially been rebranded to RoboShellGuard. This change was necessary to address name collision and ensure the project is easily discoverable through search engines, as the generic "ShellGuard" name is too common and could cause confusion. What this means for you: All features remain the same, but the project is now uniquely positioned in the automation security space. Key changes to note: The primary repository is now located at [https://github.com/robokeys/roboshellguard], and installation commands will use the new roboshellguard name.

The new name RoboShellGuard is unique and so should be much easier to search. I have just renamed ShellGuard to RoboShellGuard to make it easier to find. There are no other changes yet, though I have some exciting plans.
